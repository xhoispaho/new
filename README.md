FilmatEmi

Ky është një projekt i thjeshtë për një website filmash.

Në këtë faqe përdoruesit mund të shohin disa kategori filmash dhe një listë me filmat më të preferuar.

Teknologjitë e përdorura:

HTML
CSS
Bootstrap
Në faqe janë përfshirë:

Navbar me kategori filmash
Search bar për kërkim filmash
Seksioni "Rreth FilmatEmi"
Lista e zhanreve të filmave
Lista e filmave më të preferuar
Ky projekt është ende në fazat e para dhe do të përmirësohet më shumë me kalimin e kohës.
